# Sistema de Gestión de Alumnos

## Descripción
Aplicación de consola en Java que procesa las notas de alumnos desde un archivo y ofrece diversas funcionalidades de análisis y visualización mediante un menú interactivo.

## Funcionalidades
- **Nota media de cada alumno**: Calcula y muestra la media de cada estudiante
- **Listar aprobados**: Muestra solo alumnos con media >= 5
- **Listar ordenado**: Alumnos ordenados por nota media (de mayor a menor)
- **Guardar resultados**: Exporta las medias a un archivo "resultado.txt"
- **Menú interactivo**: Navegación mediante opciones numéricas

## Tecnologías
- Java
- HashMap / ArrayList
- BufferedReader / FileReader
- BufferedWriter / FileWriter
- Colecciones y ordenamiento
- Manejo de excepciones IOException
